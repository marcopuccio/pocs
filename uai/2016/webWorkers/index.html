Puccio Marco

Hacer click en "Comenzar" para empezar usar aplicación. Este botón, habilitará el formulario de edicion y lanzará los webworkers. Una vez presionado, este botón se ocultará y habilitará el boton de Terminar. Una vez presionado este último, no se podrá usar la aplicación hasta recargar la ventana.

El webworker "date.js" imprime la fecha en cada evento "SUBMIT" del form. El webworker "feed.js" tomará los datos ingresados en el formulario convertidos en JSON por Jquery y actualizará el sector de última noticia con un segundo de demora luego de que se dispare el evento "SUBMIT".
